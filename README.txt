Dojo Bootstrap
**************

Bootstrap theme for Dojo.

Copyright (c) 2012 Martin Pengelly-Phillips
For license see LICENSE.txt

You'll need Stylus in order to compile the .styl files to css. Get it from
https://github.com/LearnBoost/stylus

Then from the root directory of the project run:

$ stylus source/dbootstrap/theme/dbootstrap.styl --include source/nib/lib/

Now open source/index.html
